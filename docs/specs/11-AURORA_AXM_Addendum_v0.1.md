## ATR Extension Matrix (AXM)

Purpose:
- Is this stock too extended to buy?

Formula:
axm10 = (close - ema10) / atr14
axm21 = (close - ema21) / atr14
axm50 = (close - sma50) / atr14
axm200 = (close - sma200) / atr14
